# darkflare

v4 in the making - coming soon!

What you can expect from v4:

- JavaScript/TypeScript support
- Node.js v18 only
- even smaller and faster
- [wjson](https://github.com/azurydev/wjson) built-in
- support for [React](https://github.com/facebook/react)
- [TypeBox](https://github.com/sinclairzx81/typebox) built-in
- custom testing library
- built-in linting
- various implementations of popular packages
- [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0)
- [changesets](https://github.com/changesets/changesets) for changelog/publishing
- **beat itty-router, worktop, hono, and others in performance!**
